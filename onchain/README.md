# msid onchain

This repository contains the MetaSoccerID smart contract, which is used for registering and managing usernames on the MetaSoccer platform.

## Deployment Information

The MetaSoccerID contract has been deployed on the Skale Nebula network at the following address:

`0xB0bbe663E7b528568cbD38047bc34ca1ec2fFEe0`

## Getting Started for Developers

1. Clone the repository
2. Install dependencies:
   ```
   yarn install
   ```
3. Set up your environment variables in a `.env` file (see `.env.example` for required variables)

## Useful Commands

### Compile the contract

To compile the smart contract, run:

`yarn compile`

### Deploy the contract

To deploy the smart contract, run:

`yarn deploy`
